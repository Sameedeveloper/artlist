# artlist
An app for search globally recognized artists, see their social links and upcoming event dates along with ticket links.

## Running Requirements

1) Node Installed.
2) After moving the repo to your local machine, open the project project in VS Code/CMD , type 'npm install -i' to get all dependencies. Once that is finished, type 'npm run start' to run the project in a browser window.


This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts
